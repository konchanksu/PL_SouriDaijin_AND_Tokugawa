動作環境

  • ハードウェア
    • Macintosh シリーズ
  • OS
    • MacOS Sierra(10.12)以上
  • プログラミング言語
    • JDK 16以上
    • Python 3.8以上

ダウンロード方法

GitHubのページからクローンしてください。

ディレクトリの移動

クローンしてきたディレクトリ(PL_SouriDaijin_AND_Tokugawa)へ移動してください。 
  $ cd PL_SouriDaijin_AND_Tokugawa

  $ ls | sort
  CSV2HTML_by_Java
  CSV2HTML_by_Python
  README.md
  SouriDaijin_AND_Tokugawa_Document

インストール・実行方法

Java版

Javaのプログラムが入っているディレクトリ(CSV2HTML_by_Java)へ移動してください。 
  $ cd CSV2HTML_by_Java

  $ ls |sort
  CSV2HTML.app
  Makefile
  PreparePMD.sh
  build.xml
  csv2html
  csv2html.mf
  junit
  pmd
  pmd-bin
  utility.jar
make installコマンドを実行してください。 
  $ make install
  env LC_ALL=ja_JP.UTF-8 ant install
  Picked up JAVA_TOOL_OPTIONS: -Dfile.encoding=UTF-8
  Buildfile: ~/PL_SouriDaijin_AND_Tokugawa/CSV2HTML_by_Java/build.xml

  ---------------8<-----------------------------8<-------------------

  BUILD SUCCESSFUL
  Total time: 3 seconds
同じディレクトリにある(CSV2HTML.app)を実行してください。 
  $ open CSV2HTML.app

Python版

Pythonのプログラムが入っているディレクトリ(CSV2HTML_by_Python)へ移動してください。 
  $ cd CSV2HTML_by_Python

  $ ls |sort
  CSV2HTML.app
  CSV2HTML.py
  MANIFEST.in
  Makefile
  README.txt
  csv2html
  pylintrc.txt
  setup.py
make installコマンドを実行してください。 
  $ make install
  mkdir CSV2HTML.app/Contents/Resources/Python/
  cp -p -r CSV2HTML.py csv2html CSV2HTML.app/Contents/Resources/Python/
同じディレクトリにある(CSV2HTML.app)を実行してください。 
  $ open CSV2HTML.app
